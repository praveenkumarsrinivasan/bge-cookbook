# Meat Temperature Guide

## Overview

A quick-reference chart for safe internal temperatures and doneness targets.

## Process

Use the chart below when checking internal temperatures.

- Smoke: 150-250 F (66-121 C)
- BBQ:   250-350 F (121-177 C)
- Grill: 350-450 F (177-232 C)

![Meat Temperature Guide](../../assets/images/Cooking_Temp_Guide.jpg)

## Notes

- Always verify doneness with a reliable instant-read thermometer.
